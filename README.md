## üöÄ Features

### üß† **AI-Powered Estimation**
- Instantly calculates **Total Construction Cost** based on area and floors.
- Breaks down material requirements: **Steel (Tons), Cement (Bags), Sand (Tons)**.
- Uses **Real-Time Math Logic** for precision (no hallucinations).

### üìÖ **Smart Scheduling (Groq Cloud)**
- Uses **Llama-3.3-70b-versatile** via Groq Cloud to generate a **step-by-step construction timeline**.
- Visualizes the schedule using a **Gantt Chart** and a **Vertical Execution Flow**.

### üé® **Interactive Blueprints**
- Automatically generates a **2D Architectural Layout** (SVG) based on the number of floors.
